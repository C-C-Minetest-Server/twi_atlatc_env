# 1F616EMO Survival Server LuaATC Environments

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/C-C-Minetest-Server/twi_atlatc_env)

This repository comes with the source code of three [LuaATC Environments](https://advtrains.de/wiki/doku.php?id=usage:atlatc:environment):

* `1F616EMO`: Legacy station track system (v2) that integrates with `PIS_v3`; independent utilities.
* `PIS_v3`: Pluggable Passenger Information System (version 3)
* `STN_v3`: Station Track System, version 3

Check their documentations and Makefiles for more information.
